安装

本项目是基于create-react-app脚手架改造而来，因此执行如下命令即可

$ npx create-react-app my-app --scripts-version steer-react-scripts
# 或者
$ yarn create react-app my-app --scripts-version steer-react-scripts

安装完成后，开始开发你的页面吧开发文档
